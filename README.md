1. Download and extract
2. Open command prompt (Win + cmd + Enter)
3. Type cd "C:\Users[REPLACE WITH ACTUAL PATH TO FOLDER]\btc_market_maker-main\btc_market_maker-main"
4. Type py -m pip install -r requirements.txt

1. Type py main.py
